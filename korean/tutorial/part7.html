Part 7: 스크립트 이해하기

이번엔 스크립트에 쓰여있는 코드에 대한 설명을 하겠습니다.

코드

코드 설명
pos(x,y) 모니터 좌표 (x,y)로 마우스 커서를 이동합니다.
mp1 왼쪽 마우스 버튼을 누릅니다.
mr1 눌렀던 왼쪽 마우스 버튼을 뗍니다.
mp2 오른쪽 마우스 버튼을 누릅니다.
mr2 눌렀던 오른쪽 마우스 버튼을 뗍니다.
mp3 마우스 휠을 누릅니다.
mr3 눌렀던 마우스 휠을 뗍니다.
wheel(integer) wheel(-1)은 마우스 휠을 돌려서 페이지를 한 칸 올린 것과 같은 행동을 합니다.
wheel(1)은 마우스 휠을 돌려서 페이지를 한 칸 내린 것과 같은 행동을 합니다.
kp(key code) 지정한 key code의 키보드 키를 누릅니다.
key code 리스트는 여기를 클릭하여 java.awt.event.KeyEvent 항목에서 찾을 수 있습니다.
kr(key code) 눌렀던 키보드 키를 뗍니다.
key code 리스트는 여기를 클릭하여 java.awt.event.KeyEvent 항목에서 찾을 수 있습니다.
sleep(integer) 지정한 시간 동안 실행을 잠시 멈추게 됩니다.
시간의 단위는 millisecond입니다.
예를 들어, sleep(1000)은 1초를 쉬게 됩니다.
위 내용을 토대로 스크립트를 직접 수정할 수 있습니다.
예를 들면, 
mp1
sleep(auto1)
mr1
sleep(auto2)
위 코드는 오토 클릭 기능을 하게 됩니다.
(Loop 부분에 'i'를 적으셔야 합니다)
