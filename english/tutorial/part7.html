Part 7: Understanding the script

This tutorial provides more information about the script.

Codes

Code Meaning
pos(x,y) Move mouse cursor to the specified (x,y) position of your screen.
mp1 Left Mouse Press
mr1 Left Mouse Release
mp2 Right Mouse Press
mr2 Right Mouse Release
mp3 Mouse Wheel Press
mr3 Mouse Wheel Release
wheel(integer) wheel(-1) is a same action as page up by 1 unit using mouse wheel.
wheel(1) is a same action as page down by 1 unit using mouse wheel.
kp(key code) Press specified key code on your keyboard.
You can see the list of key codes on java.awt.event.KeyEvent section from here.
kr(key code) Release specified key code on your keyboard.
You can see the list of key codes on java.awt.event.KeyEvent section from here.
sleep(integer) It will be idle for specified amount of time.
The unit of time is millisecond.
For example, sleep(1000) means idle for 1 second.
Once you understand how script works, you can edit your script manually(without recording).
For example, 
mp1
sleep(auto1)
mr1
sleep(auto2)
This script will simulate auto-clicking.
(Make sure you put 'i' on Loop section)
