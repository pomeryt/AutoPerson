Part 4: Main Page

In this tutorial, the main page will be explained.
The image below shows the main page.

Meanings of each number

Number Meaning
1 It shows running status.
"Ready" means no script is running.
If the color is green, then it means a file is running.
2 It shows the start key of script.
The script will run when you press the start key on your keyboard.
If you press the same start key again while a script is running, then it will stop the script.
If you press the different start key while a script is running, then it will stop the current script and run new script.
3 It is a search bar of the script list.
4 It is a script list.
It displays all files in the script folder.
It sorts scripts in alphabetical order.
5 Button for creating a new script.
6 Button for editing a script.
Select a script you want to edit from the list before pressing the button.
7 Button for setting a start key of the script.
8 You will go to the Setting page when you press this button.
9 Refresh button for the script list.
10 You will go to the About page when you press this button.
11 Button for removing a script.
